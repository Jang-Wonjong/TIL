목록

- 순서가 없는 목록 : '-' 하이픈 하나
- 들여쓰기 하면 자동으로 다음 목록 생성
  - tab해서 다음 단으로 들어가요
  - 엔터쳐서 인어나가다가
- shift + tab으로 상단으로 나올 수 있다
- 목록을 그만 쓰고 싶으면, 엔터 2번

1. 순서가 있는 목록
2. 얘도 엔터 누르면 다음 숫자 만들어짐
   1. tab 누르면 한단 안으로 들어가구
   2. 내용 쓰다가
3. shift + tab
4. 끝낼 때도 동일
